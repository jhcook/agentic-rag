@Workflow ImplementFeature
Participants: @Engineer, @Architect, @Sentinel, @QA, @Scribe

Steps:
1. @Engineer:
   - Implement the requested feature.
   - Explicitly state how they obey **Global Standards**.

2. @Architect:
   - Review the proposed changes for architecture & boundaries.

3. @Sentinel:
   - Review only for security, secrets, and privacy issues.

4. @QAEngineer:
   - Review only for test coverage, error handling, and reliability.

5. @Scribe:
   - Review docs, naming, and change communication.

A change is ready when all roles return `VERDICT: APPROVE`.
